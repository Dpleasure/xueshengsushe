# 学生宿舍管理系统 - 快速测试指南

## 🚀 启动系统

```bash
# 1. 进入项目目录
cd xueshengsushe

# 2. 安装依赖（如果还没安装）
npm install

# 3. 启动开发服务器
npm run dev

# 4. 在浏览器中打开
# http://localhost:3000 或 http://localhost:3002
```

## ⚠️ 常见问题排查

### 问题 1：数据添加后不显示
**解决方案：**
- 检查是否点击了"确定"按钮
- 检查是否有"添加成功"提示
- 如果没有提示，检查浏览器控制台是否有错误

### 问题 2：删除后数据还在
**解决方案：**
- 确认点击了确认对话框的"确定"
- 检查是否有"删除成功"提示
- 刷新页面看数据是否真的删除了

### 问题 3：页面无法访问
**解决方案：**
- 检查开发服务器是否正常运行
- 检查端口号（可能是 3000、3001 或 3002）
- 检查浏览器控制台是否有错误

### 问题 4：注册后无法登录
**解决方案：**
- 当前是纯前端项目，注册的数据不会保存
- 登录时可以输入任意账号密码
- 如需真实的注册登录，需要集成后端 API
