# 学生宿舍管理微信小程序

这是在不修改原后端和 Web 前端的基础上新增的学生端微信小程序。

## 运行前准备

1. 启动后端 Spring Boot 服务，确保接口地址可访问：

   ```text
   http://127.0.0.1:8080
   ```

2. 打开微信开发者工具，选择“导入项目”。

3. 项目目录选择：

   ```text
   C:\Users\17336\Desktop\xueshengsushe-master\miniprogram
   ```

4. AppID 可以先选择测试号或使用游客模式。

5. 在微信开发者工具中勾选：

   ```text
   不校验合法域名、web-view、TLS 版本以及 HTTPS 证书
   ```

## 后端地址

本地开发默认地址在 `app.js`：

```js
baseUrl: 'http://127.0.0.1:8080'
```

如果要用手机真机预览，需要改成电脑的局域网 IP，例如：

```js
baseUrl: 'http://192.168.1.10:8080'
```

同时手机和电脑必须连接同一个 Wi-Fi。

## 已实现功能

- 学生登录
- 首页住宿信息
- 我的个人信息
- 我的报修列表与提交报修
- 我的换宿记录与提交换宿申请
- 我的来访记录与提交来访登记

## 说明

第一版只做学生端，不做管理员端；报修第一版不上传图片，只提交文字描述和宿舍信息。
